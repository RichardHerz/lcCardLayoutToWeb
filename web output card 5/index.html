The outer group of this layout is the largest you will be able to fit in the width of an iPhone browser, width = 958, with a css margin of 2px.

The css pixel width of an iPhone 5s is 980, whereas the physical pixel width is 640.

The outer group of a simulations (div) should have its css position changed to relative.

Buttons and other controls should be made larger than the default in LiveCode in order to manipulate more easily in web browsers.
